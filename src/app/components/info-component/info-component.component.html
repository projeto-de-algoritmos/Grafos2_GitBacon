SIGAA O SIGAA, Sistema Integrado de Gestão de Atividades Acadêmicas, é o sistema de administração da vida acadêmica do estudante, da Universidade de Brasília (UnB), nele é possível que os estudantes realizem diversas atividades como emissão de histórico escolar, atestado de matrícula, declaração de vínculo com a instituição, matrícula. Dessa forma, nele estão contidas informações relativas às grades curriculares e disciplinas de cursos da UnB, juntamente com seus pré-requisitos e co-requisitos.
Grafos Grafos são estruturas matemáticas criadas para representar relacionamentos entre objetos, assim, podendo relacionar uma informação a outra. Para isso, são utilizados nós que representam os objetos e arestas que representam uma ligação entre os objetos, essas ligações podem ser assimétricas, representadas por grafos não direcionados e podem ter pesos associados a elas, como é o caso de uma ligação entre duas disciplinas no SIGAA.
SIGAARush O SIGAARush foi desenvolvido com o objetivo de auxiliar os estudantes da Universidade de Brasília a descobrirem a forma mais rápida de cursar uma disciplina desejada, para isso foi utilizada a técnica de Web scraping que coleta os dados das disciplinas no sítio eletrônico do SIGAA, para descobrir qual os pré-requisitos das matéria desejada e, assim, calcular o menor caminho, no qual os pesos das arestas é a carga horária necessárias para cursar a disciplina que é pré-requisito. Entretanto não é garantido que essas disciplina possuem oferta para o semestre atual, devido os dados terem sido obtidos pela técnica de Web scrapping a partir da página do SIGAA .
